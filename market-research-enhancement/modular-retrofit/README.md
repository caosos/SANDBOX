# Modular Retrofit Research Project

Branch: `market-research-enhancement`

Folder: `market-research-enhancement/modular-retrofit/`

## Purpose

This folder captures research, product strategy, and prototype planning for an aftermarket modular retrofit system for full-size silicone/TPE dolls that were originally built with fixed internal openings.

The working thesis is:

> The insert is a component. The retrofit collar/socket system is the actual product moat.

## Files

- `PRODUCT_SPEC.md` — product architecture, safety boundaries, MVP sequence, business model, and risks.
- `MOLD_PLAN.md` — practical mold-making and bench-test path.
- `RECEIPTS_PROMPT.md` — evidence-only research prompt to validate or invalidate the market gap.

## Current Product Direction

Start with a passive mechanical retrofit system before adding powered features.

Phase order:

1. Material test blocks
2. Passive removable insert and reinforced collar
3. Sacrificial torso/damaged doll test
4. Controlled installation service
5. DIY kit only after the process is proven
6. Optional 5V powered module after mechanical system passes testing

## Power Rule

Prototype powered features must use low-voltage DC only, preferably 5V for early testing.

No 120V AC inside the product. No buried lithium battery in early prototypes.

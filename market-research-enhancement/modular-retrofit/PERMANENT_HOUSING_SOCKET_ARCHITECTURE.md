# Permanent Housing / Removable Insert Architecture

## Corrected Product Architecture

The glued-in component is not the removable insert.

The glued-in component is the permanent housing/socket.

The insert remains removable.

## Core System

1. A standardized opening is created in the fixed doll body.
2. A permanent housing/socket is bonded into that opening.
3. The housing stays attached to the doll.
4. A removable insert slides into the housing.
5. The insert can be removed for cleaning, drying, replacement, or upgrade.

## Product Roles

### Permanent Housing / Socket

This is the installed retrofit part.

It should:

- stay attached to the doll body
- reinforce the modified opening
- create a standardized inner cavity
- accept replaceable inserts
- prevent edge tearing
- provide a smooth transition from doll material to insert
- keep the insert aligned
- prevent the insert from shifting or backing out during handling

### Removable Insert

This is the replaceable cartridge.

It should:

- slide into the permanent housing
- seat securely
- be removable without damaging the housing
- be cleanable outside the doll
- be replaceable when worn
- support future passive, powered, heated, or upgraded versions

## Mechanical Design Concept

The housing should work like a bonded socket or receiver.

Possible housing features:

- outer flange bonded to doll body
- inner sleeve wall
- rear lip or retention shoulder
- tapered lead-in
- soft sealing rim
- anti-rotation key
- drain/dry access path if compatible with the design
- standardized cartridge size

Possible insert features:

- matching taper
- rear flange or pull feature
- keyed alignment slot
- retention groove
- smooth exterior surface
- replaceable cartridge body

## Best MVP Geometry

Start with one universal size class.

Suggested MVP:

- oval/rounded opening
- bonded flexible housing
- rear retention shoulder
- removable insert with matching taper
- light mechanical retention, not glue
- cleaning/drying access with insert removed

## Why This Is Better Than Gluing The Insert

If the insert is glued in, the product does not solve the cleaning problem.

If the housing is glued in and the insert is removable, the product solves:

- cleaning
- drying
- replacement
- customization
- future upgrades
- warrantyable insert replacement
- repeat purchases

## Business Model Impact

The permanent housing creates the platform.

The removable insert creates recurring sales.

Possible sales after one installation:

- replacement inserts
- softer/firmer inserts
- upgraded inserts
- powered 5V inserts
- heating inserts
- cleaning kits
- drying kits
- repair collars
- size adapters

## Installation System

The install kit should be built around this sequence:

1. Cut or core the opening using a standardized cutter.
2. Clean and prep the modified opening.
3. Bond the housing/socket into the body.
4. Let bonding/cure complete.
5. Insert the removable cartridge.
6. Test retention.
7. Remove cartridge.
8. Verify cleaning/drying access.

## Product Thesis

The housing is the retrofit.

The insert is the consumable/upgrade.

The cutter makes the opening.

The housing makes the doll modular.

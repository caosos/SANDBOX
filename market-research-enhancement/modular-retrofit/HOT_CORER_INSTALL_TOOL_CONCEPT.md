# Heated Corer / Retrofit Cutter Tool Concept

## User Concept

A thin stainless-steel formed cutter/corer, similar in concept to an apple corer, made in different sizes and angles. The tool would remove material from a fixed-opening doll so a removable insert/collar/socket can be installed.

Original rough idea:

- formed stainless steel tube/cutter
- multiple sizes
- heated before use
- inserted at a chosen angle
- core removed by twist/pull/cut
- followed by retrofit collar/socket installation

## Product Verdict

The cutting/coring concept is strong.

The customer-heated oven version is not the right commercial method.

Better direction:

- controlled installer tool
- shop-use jig
- professional service tool
- optional future certified-installer kit

The retrofit cutter should be treated like an installation tool, not a casual DIY kitchen-heated part.

## Why The Tool Matters

The retrofit system needs a repeatable opening geometry.

Freehand cutting creates:

- uneven openings
- jagged edges
- tear paths
- poor collar fit
- unpredictable insert angle
- bad customer outcomes

A standardized coring/cutter tool can create:

- repeatable socket geometry
- cleaner edge
- predictable angle
- consistent collar fit
- faster installation
- compatibility database by cutter size

## Better Tool Architecture

### Tool Body

- thin-wall stainless steel cutter tube
- oval/rounded profile, not sharp-corner geometry
- replaceable cutting edge
- depth stop
- angle guide
- handle or T-handle
- indexed size markings
- collar-size matching marks

### Size Family

Possible sizes:

- small
- medium
- large
- custom/deep

Each size should match:

- insert diameter
- collar outer diameter
- required wall thickness
- doll material type
- installation angle

### Heat Control

Do not rely on uncontrolled customer oven heating as the main commercial workflow.

Better options:

1. room-temperature mechanical corer for soft material
2. controlled-temperature bench heater
3. heated blade/corer station with temperature display
4. disposable sharp coring edge plus non-heated install method

### Depth and Angle Control

The tool needs a depth stop and angle guide.

Without those, the user can cut too deep, hit skeleton, cut off-angle, or create an opening the collar cannot seal.

## Installation Workflow Concept

1. Identify doll material and target opening.
2. Measure required insert/collar size.
3. Select matching cutter/corer.
4. Mark angle and depth.
5. Use guide jig or depth stop.
6. Remove core cleanly.
7. Inspect cut edge.
8. Install reinforced collar/socket.
9. Seat removable insert.
10. Pull-test, clean-test, and inspect.

## MVP Tool Test

Build tool prototypes from:

- stainless tubing
- sharpened thin-wall edge
- welded/brazed handle
- clamp-on depth stop
- angle guide plate

Test on:

- TPE sample block
- silicone sample block
- damaged doll/torso only after sample blocks

Measure:

- edge quality
- force required
- material tearing
- core removal cleanliness
- repeatability
- collar fit
- insert angle

## Business Model Implication

This tool supports three business paths:

1. Controlled in-house retrofit service
2. Certified installer kit
3. Advanced DIY kit after the process is proven

Do not lead with casual DIY.

Lead with controlled installation and use the cutter to create repeatable results.

## Core Takeaway

The cutter/corer is a key part of the retrofit system.

The business should not sell a random heated kitchen-style tool.

The business should sell or use a calibrated retrofit cutter that makes the collar/socket installation repeatable.

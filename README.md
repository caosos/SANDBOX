# CAOS SANDBOX

This repository is the working storage area for Michael's build ideas, documents, prototypes, scripts, plans, and project artifacts.

## Operating rule

When Michael asks Aria to create something project-related, default to storing the working text/code here unless he says otherwise.

## Folder map

```text
/Senior-Mobility-Repair/       Senior mobility repair, outreach, ads, scripts, call sheets
/CAOSCare/                     Senior-care AI prototype, room systems, Home Assistant, device notes
/BringYourAI/                  Business visibility engine, local AI/business launch material
/Business-Visibility-Engine/   Listings, SEO, AI visibility, audit/report concepts
/Brookdale-Tools/              General non-confidential workflow templates only
/Scratchpad/                   Raw ideas, quick captures, unsorted notes
/_Ops/                         Repo operating rules, indexes, templates
```

## Hard boundaries

Do not store passwords, API keys, resident data, private medical information, Brookdale confidential material, HR data, family private data, or anything that should not be in a public GitHub repository.

If an artifact contains sensitive information, create a sanitized template here instead.

## Workflow

1. Capture raw idea.
2. Sort into project folder.
3. Convert into usable artifact: plan, ad, SOP, tracker, script, checklist, README, CSV, or code.
4. Keep a short decision log when the direction changes.
5. Split mature projects into their own repository later.

# AI-Assisted Dating / Relationship Compatibility Service

Status: **primary current project**.

## Core thesis

This is not a swipe app. It is a truthful, AI-assisted compatibility and relationship-screening service.

The service helps people build a deep profile with AI, clarifies standards and hard-no criteria, and provides curated matches instead of endless browsing.

## Product direction

- Fewer matches, higher trust.
- One good match matters more than high match volume.
- AI helps build an honest profile instead of letting users perform or posture.
- Pictures are not the first filter.
- Real compatibility should be screened before emotional attachment.
- Users may accept or reject a match, but the system's value is that the match was selected for a reason.
- Local-first launch: Conway / Faulkner County is a likely starting area.
- Premium positioning is acceptable if the value is real.

## What not to do

- Do not make a swipe-right/swipe-left clone.
- Do not make a hookup site.
- Do not start with UI before criteria.
- Do not assume generic dating-app best practices apply.
- Do not reduce the system to a personality quiz.
- Do not lead with photos as the primary filter.
